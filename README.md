# Planning Poking

## Tech used
### Back
- Deno
- Oak
- MongoDB (optional)
### Front
- Angular
- Angular Material
- TailwindCSS

## Installation
You need to install npm and deno to run this application. Use these commands to check that they are installed:
-for npm : npm -v
-for deno : deno -V
Both should print the version

## How to run it
In the root folder, execute :
```
deno task front
deno task back
```
